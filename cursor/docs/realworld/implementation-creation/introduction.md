# Introduction

Conduit is a social blogging site (i.e. a Medium.com clone). It uses a custom API for all requests, including authentication.

Otherwise:

3. Read the following sections: [expectations](./expectations.md) and [features](./features.md) for a better understanding of this project
4. [Fork our starter kit](https://github.com/gothinkster/realworld-starter-kit)

Happy coding!
